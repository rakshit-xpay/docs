Welcome to the xPay API documentation! Here, you'll find everything you need to get started with integrating our payment solutions into your application.

## Base API URL

The base URL for all xPay API requests is: 

```bash
https://api.xpaycheckout.com
```

## Available APIs

Hereâ€™s a list of the key APIs you can integrate with:

- **Create Intent**

  - **Endpoint**: `/payments/create-intent`
  - **Description**: Initiates a payment intent with necessary transaction details.
  - **Detailed Documentation**: [Create Intent API](/endpoint/xPayAPIs/create-intent)

- **Get Intent**

  - **Endpoint**: `/payments/get-intent/{xIntentId}`
  - **Description**: Retrieves the status and details of a payment intent.
  - **Detailed Documentation**: [Get Intent API](/endpoint/xPayAPIs/get-intent)

- **Payment Links**

  - **Endpoint**: `link/merchant/generate-link`
  - **Description**: Generates a payment link for a specific amount.
  - **Detailed Documentation**: [Payment Links API](/endpoint/xPayAPIs/generate-link)

- **Create Intent with Tax**

  - **Endpoint**: `/payments/create-intent-with-tax`
  - **Description**: Initiates a payment intent including tax calculations.
  - **Detailed Documentation**: [Create Intent with Tax API](/endpoint/salesTaxAPIs/create-intent-with-tax)

- **Get Intent with Tax**
  - **Endpoint**: `/payments/get-intent-with-tax/{xIntentId}`
  - **Description**: Retrieves the status and tax details of a payment intent.
  - **Detailed Documentation**: [Get Intent with Tax API](/endpoint/salesTaxAPIs/get-intent-with-tax)
