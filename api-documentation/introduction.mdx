Welcome to the xPay API documentation! Here, you'll find everything you need to get started with integrating our payment solutions into your application.

## Base API URL

The base URL for all xPay API requests is: 

```bash
https://api.xpaycheckout.com
```

## Available APIs

Hereâ€™s a list of the key APIs you can integrate with:

| API Name                    | Endpoint                              | Description                                             | Documentation                               |
|-----------------------------|---------------------------------------|---------------------------------------------------------|-----------------------------------------------------|
| **Create Intent**           | `/payments/create-intent`             | Initiates a payment intent with necessary transaction details. | [Create Intent API](/endpoint/xPayAPIs/create-intent) |
| **Get Intent**              | `/payments/get-intent/{xIntentId}`    | Retrieves the status and details of a payment intent.   | [Get Intent API](/endpoint/xPayAPIs/get-intent)       |
| **Payment Links**           | `/link/merchant/generate-link`         | Generates a payment link for a specific amount.        | [Payment Links API](/endpoint/xPayAPIs/generate-link) |
| **Refund**                  | `/payments/merchant/refund`            | Processes a refund for a specified payment intent.     | [Refund API](/endpoint/xPayAPIs/refund)|
| **Cancel Payment**     | `/payments/cancel/{xIntentId}`                 | Cancels a payment.                                | [Cancel Payment API](/endpoint/xPayAPIs/cancel-payment) |  
| **Create Subscription**     | `/subscription/create`                  | Create a subscription for a customer.                  | [Create Subscription API](/endpoint/xPayAPIs/create-subscription) |
| **Get Subscription**        | `/subscription/get-subscription/{subscriptionId}`   | Retrieves the details of a subscription.  | [Get Subscription API](/endpoint/xPayAPIs/get-subscription) |
| **Cancel Subscription**     | `/subscription/merchant/cancel`                 | Cancels a subscription.                                | [Cancel Subscription API](/endpoint/xPayAPIs/cancel) |
| **Subscription Links**     | `/subscription/link/merchant/generate-link`                 | Generates a subscription link for a specific amount.                                | [ Generate Subscription Link](/endpoint/xPayAPIs/subscription-link) |
